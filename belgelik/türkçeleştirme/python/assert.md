# assert() Kullanımı

**AÇIKLAMA**
`assert()`fonksiyonu,belirli bir ifadenin doğru olup olmadığını sınar.

**GİRDİ**
```
def avg(marks):
    assert len(marks) != 0
    return sum(marks)/len(marks)

mark1 = []
print("Average of mark1:",avg(mark1))
```

